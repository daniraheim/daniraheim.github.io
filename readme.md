2. I built a gym website for DaniFlow Studio.  The webpage allows users to see some key features offered by the gym.  It also allows the users to fill out a form to be submitted to join the gym. Also, the users can check out the social media pages for the gym.

3. Key components used in building this page would include, HTML, CSS, Bootstrap, and JavaScript.  The page includes a carousel, embedded Google map, formspree.io, and a navigation menu.

4.  Users can open the webpage, read about the features on the gym, check out the images of the inside of the gym.  Then, the users can see the pricing and join today.

5. Future plans for the project will include embedded videos of the gym, testimonials from existing members, and a chat option to chat with an employee live for any questions or concerns.
